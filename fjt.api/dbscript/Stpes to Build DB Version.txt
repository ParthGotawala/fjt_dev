Take Care Steps for db version execution.
--------------------------
1. Create your own script first in one file.
2. Check selected db on you are working.
3. Copy whole script in editor (must check no syntax error).
4. Take latest before adding new script in DB_change file.
5. Paste formatted string in DB_change file.
6. Change build number for your script. 
7. Must take care while two person change at same time, last one have to do change in build number.
8. Before build must rerun api project as we have taken latest.
9. Check at locally first with another db, if it get executed successfully than do check-in.
10. For your db comment your current script and execute so that db version get executed for your build number.
